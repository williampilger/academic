## Tabelas

| **Tag** | **Função** |
| --- | --- |
| `<table>` | Cria uma tabela |
| `<tr>` | Linha de uma tabela |
| `<th>` | Coluna de uma tabela |
| `<td>` | Coluna de cabeçalho |

Exemplo:

******

**Codigo**

```html
<table>
    <tr>
        <th>Nome</th>
        <th>Telefone</th>
    </tr>
    <tr>
        <td>Renato</td>
        <td>(51) 99999 9999</td>
    </tr>
    <tr>
        <td>William</td>
        <td>(51) 00000 0000</td>
    </tr>
</table>
```

**Resultado**

<table>
    <tr>
        <th>Nome</th>
        <th>Telefone</th>
    </tr>
    <tr>
        <td>Renato</td>
        <td>(51) 99999 9999</td>
    </tr>
    <tr>
        <td>William</td>
        <td>(51) 00000 0000</td>
    </tr>
</table>
